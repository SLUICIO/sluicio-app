# Sluicio API (v1)

> Self-hosted integration monitoring on OpenTelemetry. Every endpoint returns JSON and is RBAC-filtered to the caller's token.

Auth: `Authorization: Bearer <token>` — a personal access token or a service-account token (Settings → Service accounts). Time windows: `?range=1h|24h|7d` or `from/to` ISO pair.

Example: `curl -H "Authorization: Bearer $TOKEN" https://<cell>/api/v1/services?range=24h`

Machine spec: /api/v1/openapi.json · Interactive: /api/docs · AI agents: prefer MCP at POST /api/v1/mcp (same auth, curated read tools).
